## Introduction to Python Dictionaries
## What is a dictionary? A dictionary is one of the main data structure (structures we use to store data e.g list, sets, tuple etc) we use in python which is a lot like a list, with one major difference: Items in a **list** are accessed using **numbers.** Items in a **dictionary** are accessed using strings of **text.** In other words, a dictionary is a data structure in which each piece of data (a *value*) is associated with a word (a *key*).
